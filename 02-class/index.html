Incorporar Clases Dinamicas con HTML

El siguiente cuadro de texto es dinámico, por lo que podremos cambiar el color de fondo, de acuerdo a la clase que coloquemos en el siguiente input. Algunos de los colores válidos son:

  • bg-primary
  • bg-secondary
  • bg-dark
  • bg-danger
  • bg-warning

Cuando el estado de 'color' se encuentre en FALSE, no se podrá visualizar el contenedor de fondo verde. Si cambiamos el estado a TRUE, mediante la actuacion de un evento @click, modificaremos el estado para poder ver el contenedor de fondo verde.
